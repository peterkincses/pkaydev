Gousto.co.uk

February 2014

London, UK

Platform: Twitter Bootstrap

Used: Twitter Bootstrap, Html5, Css3, lots and lots of jQuery

I was approached by the back end development team at Gousto to create a bunch of prototypes which will form the front end structure of the new responsive Gousto site.

The designs were based on the existing "one-size-(not)fits-all" site however I've been asked to use the Twitter Bootstrap framework so that the site remains useable even in smaller screens.

While this was my first commercial project with Bootstrap, I felt that having worked with a Magento type MVC framework gave me a massive help in understanding how to modify it for the site's needs.

I was also pleasanlty suprised by the detailed documentation and how many features have been packaged with Bootstrap.
